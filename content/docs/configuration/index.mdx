This section is where the setup lives.

Some of it is one-time setup. Some of it changes as you go.

Main areas:

- [Providers and models](/docs/configuration/providers-and-models)
- [Search, voice, images, and videos](/docs/configuration/search-voice-and-media)
- [Memory, security, and data](/docs/configuration/memory-security-and-data)
- [Integrations and MCP](/docs/configuration/integrations-and-mcp)
- [Skills](/docs/configuration/skills)

The settings surface is wide because Sentinel handles more than chat.

It also handles memory, approvals, media generation, integrations, and local state.
